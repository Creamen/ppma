# PHP Password Manager (ppma) [![Build Status](https://travis-ci.org/pklink/ppma.svg?branch=master)](https://travis-ci.org/pklink/ppma)

## Installation

```sh
cp .env.example .env
vim .env
composer install
bower install
php artisan migrate
php -S localhost:8080 -t ./public/
```
